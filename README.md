## O que eu entrego

- **Automação sustentável** — Playwright, Robot Framework, Cypress, K6 e Vitest integrados a CI/CD com Page Objects, Screenplay Pattern e Contract Testing. Pipeline Docker-based para paridade 100% local/CI.
- **Observabilidade real** — Lighthouse para web, K6 para carga e métricas que separam *"os testes passaram"* de *"o usuário conseguiu comprar"*.
- **AI no pipeline de QA** — Agentes que geram Gherkin a partir de User Stories, analisam falhas de E2E e aplicam LLM-as-a-Judge em CI/CD. IA como multiplicador, não substituto.
